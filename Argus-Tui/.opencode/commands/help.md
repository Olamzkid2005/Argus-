---
Argus security assessment commands:
- `/assess <target>` — Run full assessment via `argus assess <target>`
- `/recon <target>` — Recon only via `argus assess <target> --deterministic`
- `/doctor` — Health checks via `argus doctor`
- `/status` — System status
- `/findings` — Browse findings
- `/engagements` — List engagements
- `/report <id>` — Generate report
- `/tools` — Show MCP tools
- `/workflows` — Show workflows
- `/workspace` — Open workspace
